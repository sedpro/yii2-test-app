Yii 2 Test Application
============================

INSTALLATION
------------

### Install from git:
`git clone git@github.com:sedpro/yii2-test-app.git`

CONFIGURATION
-------------

### Database

Edit the file `config/db.php` with real data, for example:

```php
return [
    'class' => 'yii\db\Connection',
    'dsn' => 'mysql:host=localhost;dbname=yii2basic',
    'username' => 'root',
    'password' => '1234',
    'charset' => 'utf8',
];
```
